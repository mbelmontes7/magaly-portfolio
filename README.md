# Magaly Portfolio

A personal portfolio site, built with React, Vite, Tailwind CSS, Framer Motion, and Lucide React. The site showcases an interactive personal brand experience with animated sections for the hero, about, projects, and contact areas.

## Features

- Animated hero section with typewriter text and a custom laptop/video visual
- Smooth fixed section navigation for Home, About Me, Projects, and Contact
- Interactive about section with an animated skill envelope and personal coding card
- Project shelf UI with selectable project details and external links
- Contact section with an animated phone interaction, email, and social links
- Firebase hosting configuration for single-page app deployment

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- Firebase Hosting
